2018-5-28

发表于 2018-05-28 | 分类于 我的青春流水账

       5月28号了，很久没这么早就坐到电脑前了，果然人还是要找点事情做，不然闲下来就容易瞎想。2018年的这个5月又快结束了，怎么说呢，今年的这个5月有苦有甜。第一件事就是我毕业了，毫无征兆大学就这么结束了，时间可过的真快啊，从去年11月份开始算，我居然都工作大半年了。从最开始的2200，涨到了现在的5500，挺开心的，这个数字也算是给我大学时光做了个交代吧，虽然有时候说钱很俗，但是钱真的是个好东西，我家最近把房子买了，爸妈幸苦了大半辈子攒下的积蓄这次全部投进去了，前天，我问我妈妈借8千块钱，我妈说现在拿不出，家里现在没钱了，当时我真的是震了一下，爸妈挺不容易的，白手起家操劳了大半辈子，有一次我无意听到我爸妈他们说：“现在还拼几年，你崽就没有那么大的压力”，我真的心里起了波澜，我以后一定会好好的孝顺他们。这个月还举行了专升本考试，非常谢谢HWX，当初我也是多么的想通过这次考试，人长大了很多思想也就不同了，离开长沙的理由是有很多的，但是有时候留下来的理由有一个就够了。刚很认真的看了Sun写的几篇文章，是的，好的陪伴总是在身边，简简单单几个字我真的思考了蛮久，挺有感触的，其实我心里是有很多想法想要说出来的，可是真正坐到电脑前才发现自己什么都说不出。我现在知道，我并不是个佛性的人，也没有人能做到佛系的。上周，她工作成腰肌劳损了，真的很心疼的，这个月很多开心的时刻也是与她有关，她想要一个水果店，我希望我能努点力，给她一个，算是画个大饼吧。6月要来了，公司新开了一个区块链的项目，最近跟着两个腾讯出来的一起写DemoAPP，真的是有点累，不过还好，是有收获的。以后的日子应该要更加努力，为了爸妈，为了家里人，为了她。

那一下感觉真的好好。




有那么几下，我觉得我好垃圾。
# 我的青春流水账
