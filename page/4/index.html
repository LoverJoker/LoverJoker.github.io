数据库-oracle基础

发表于 2017-05-17 | 分类于 数据库

人的一切痛苦，本质上都是对自己的无能的愤怒。

基础

  • nal(a,b)当a为Null的时候取值b，当a不为null时。
  • select * from money is null ,—–查询money字段为空值的记录
  • oracle 别名 和Mysql 一样.
  • sekect distinct money from emp —查询emp表中所有的钱的数量，去除重复。distinct这个关键字表示去重.
  • 在Oracle中单引号表示日期和字符，双引号表示列的别名。
  • between a and b 查询大于等于a小于等于b的值
  • like 模糊查询 %表示任意长度的任意字符串 _表示任意字符的一个字符串, \表示转义字符.
  • 在and语句中，where解析顺序 右—->左
  • order by 后面 +列、表达式、别名、序号 + 排序规则 asc desc
  • order by 作用于后面所有的列，先按照第一个列排序，再后面的列，desc降序 制作用于离他最近的列。
  • nulls last 表示如果有空值，空值放在最后 select * from emp order by money desc nulls last 升序时没有问题。因为在oracle数据库中，null最大
  • dual 为oracle中没有意义的空表
  • 去掉delect产生后的碎片 alter table <表名>mover 提高查询速度
  • rownum 行号 select rownum from a ， rownum永远按照默认（即使排完序后行号也不变）的顺序，rownum只能使用小于，小于等于，不能用其他。
  • top-n问题：select f from (select from a order by money desc) where rownum<=3.
  • 所有包含于select列表中，而未包含于组函数中的列，都必须包含于GroupBy子句中。包含在Group子句中的列，不一定要包含在select中。
  • 多个列的分组：先按照第一个列分组，如果相同，再按照第二个列分组，以此类推。
  • having语句：从分组的基础上，过滤。select b avg(c) from table group by b having avg(c)>2000.和where的区别是，having可以使用组函数（多行函数）。
  • group by rollup(a,b) : 报表常用。

2017-4-23

发表于 2017-04-23 | 分类于 我的青春流水账

2017-4-23

       今天是2017年4月22周六，我坐在去邓胖子医院的公交车上，其实很早就想写这篇文章记录下最近的生活，奈何深知自己文笔不够，所以一直没有动手，现在正好无聊就写下吧。
       脑袋里过了一下，最近发生了很多事情。上个月我和另外两个小伙伴去参加了学校的移动开发比赛的选拔赛，比赛完第二天就通知我们被选上了，心里当然是很高兴的，我也从未看见满哥热情如此高过，当天就从家里赶到学校里面帮我们几个安排教室，请假，满哥还和我们谈了一下“男人理论”说实话当时很感动，可我已经忘记了当时满哥的原话是什么了，这个有点遗憾，就这样长达一个月的集训就开始了，刚开始借着新鲜感，我们制定了计划，每天早上7.50从寝室出发然后去吃早餐，一直在那里待到18.30。第一个礼拜我们都严格执行了这个计划，慢慢的当新鲜感一过去我们就开始慢慢的放松起来，早上8点多才起来，有几次下午2点多就走了，很是惭愧，写到这里想起一句话，虽然和现在说的这个话题没什么关系，但是我还是想写这在这里:“一直认为，所谓的新鲜感，不是和未知的人一起去做相同的事情，而是和已知的人一起去体验未知的人生。”最近养成了一个习惯，就是有时候看到喜欢的句子就会摘抄到记事本里面，希望以后
看到美景时，当别人只能说“我操，真美”，我也能说出那句“大漠孤烟直，长河落日圆”。哦，对了，现在每天都会更新下博客，把当天学的东西记录下来，现在我会在每一篇技术日志前面加上一段美句，希望自己有意没意得看一看。就这样又过了几天，满哥带我去参加了关于省赛的说明会，这不是重点，关键是中午和院长书记等一大片领导吃了饭，很荣幸但我还是不习惯那种场合，我理想的生活应该是有酒，有，算了，我不喝酒，有老友就够了。那周是我本学期第一次没有回家，第二周是清明节假期，没人管的懒散时间总是过得很快的，到了清明节放假的时候了，那天我和涂sir一起骑自行车回的家，很久没有想那样运动过了。接下来的是一段重复的日子每天吃饭睡觉，写代码，调侃Zavion。重复的日子总是无聊的，某天我从室友那里拿了一本余华写的《活着》，老实说，在我过去19年岁月里我其实是不爱看书的，因为我觉得看这种文学书好浪费时间，现在不这么想了，我一口气买了7本书，愿以后和别人讨论的时候能有个装逼的资本，增加点内涵。这篇流水账时间已经记到了现在，我也从昨天写到了今天（2017年4月23日），昨天度过了非常愉快的一天，毅少和赵泽的另外的一个同学准备到含浦去玩，我直接从一医院去步行街找了毅少，然后那里等了赵泽一下午，一直到晚上8点才到中医，晚上找了家烤鱼店，一起调侃调侃一起扯扯段子，就够了，生活本应这样。晚上我们扯段子到凌晨2点，很久没这么晚睡了，但我觉得很值得，无奈的是今天7点就要起来去学校，满哥今天会去509视察我们的情况。现在是8.20我走在去软件的路上，生活还得继续。
       希望多年以后我再次看到这篇文章时能感叹:我操，我TM还做过这么牛逼的事情？很俗但很有成就感。
