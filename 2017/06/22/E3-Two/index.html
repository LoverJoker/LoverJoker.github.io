nginx

安装需要的环境命令

  • yum install gcc-c++
  • yum install -y pcre pcre-devel
  • yum install -y zlib zlib-devel
  • yum install -y openssl openssl-devel

    安装步骤

  1. 把nginx的源码包上传到linux系统
  2. 解压缩 tar zxf nginx-1.8.0.tar.gz
  3. 使用configure命令创建一makeFile文件

  5. make

  6. make install

启动nginx

进入sbin目录

  • ./nginx 启动
  • ./nginx -s stop 关闭
  • ./nginx -s reload 刷新配置文件
  • 默认端口是80
