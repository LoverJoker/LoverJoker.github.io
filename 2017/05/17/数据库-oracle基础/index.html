数据库-oracle基础

发表于 2017-05-17 | 分类于 数据库

人的一切痛苦，本质上都是对自己的无能的愤怒。

基础

  • nal(a,b)当a为Null的时候取值b，当a不为null时。
  • select * from money is null ,—–查询money字段为空值的记录
  • oracle 别名 和Mysql 一样.
  • sekect distinct money from emp —查询emp表中所有的钱的数量，去除重复。distinct这个关键字表示去重.
  • 在Oracle中单引号表示日期和字符，双引号表示列的别名。
  • between a and b 查询大于等于a小于等于b的值
  • like 模糊查询 %表示任意长度的任意字符串 _表示任意字符的一个字符串, \表示转义字符.
  • 在and语句中，where解析顺序 右—->左
  • order by 后面 +列、表达式、别名、序号 + 排序规则 asc desc
  • order by 作用于后面所有的列，先按照第一个列排序，再后面的列，desc降序 制作用于离他最近的列。
  • nulls last 表示如果有空值，空值放在最后 select * from emp order by money desc nulls last 升序时没有问题。因为在oracle数据库中，null最大
  • dual 为oracle中没有意义的空表
  • 去掉delect产生后的碎片 alter table <表名>mover 提高查询速度
  • rownum 行号 select rownum from a ， rownum永远按照默认（即使排完序后行号也不变）的顺序，rownum只能使用小于，小于等于，不能用其他。
  • top-n问题：select f from (select from a order by money desc) where rownum<=3.
  • 所有包含于select列表中，而未包含于组函数中的列，都必须包含于GroupBy子句中。包含在Group子句中的列，不一定要包含在select中。
  • 多个列的分组：先按照第一个列分组，如果相同，再按照第二个列分组，以此类推。
  • having语句：从分组的基础上，过滤。select b avg(c) from table group by b having avg(c)>2000.和where的区别是，having可以使用组函数（多行函数）。
  • group by rollup(a,b) : 报表常用。

函数

字符类的函数

  • lower(‘A’)将字符串转小写
  • upper(‘a’)将字符串转为大写
  • initcap(‘aA’)将字符串首字母大写
  • substr(‘a’,b)从a中第b位开始取
  • instr(‘a’,’b’) 在a中查找b 查到返回位置 没查到返回0

    数值函数

  • rount() 四舍五入 round(45.926,2) = 45.93 表示在45.926中取两位小数 四舍五入
  • trunc() 截断 用法同上相似

    日期函数

  • to_char(a,格式)将a按照格式转化为字符串 比如： to_char(sysdate,’yyyy-mm-dd hhh24:mi:ss’)
  • 在日期上进行数字运算 还是为日期 数字单位是/天。
  • 两个日期可以相减，返回的是中间差的天数
  • 日期不可以相加，只可以加数字。
  • Months_Between(a,b)a,b 都是日期 这个函数返回两个日期相差的月数
  • add_months(a,b)在a的日期加上b个月
  • last_day(a)日期所在月份的最后一天
  • next_day(a，’星期四’)从a开始算，下一个星期四是多少号。

    转换函数

  • to_char(a,’b’)将a按照b的格式转化为字符串
  • to_number
  • to_date
  • wm_concat()行转列

    通用函数

  • nvl2(a,b,c)当a=null时，返回c，否则返回b
  • nullif（a,b）当a=b的时候，返回Null，否则返回a
  • coalesce(a,b,c……,z) 从左到右找到第一个不为null的值.

    条件表达式

  • case 比较通用 case a when b then c else d end。当a等于b的时候返回c，否则d。
  • decode oralce函数 decode(a,’b’,c,’e’,f,g) 别名 当a等于b的时候执行c ，当a等于e的时候执行f，否则执行g。

    查询

  • 左外连接 等号左边的表任然被包含在最后的表中 写法： where a= b(+)
  • 自连接 一张表制定两个别名，当两个表用。
  • 子查询就是select语句的嵌套。不可以在group by使用子查询。在top-n（前N名）分析问题中必须要给子查询排序。可以在主查询对策where select having from 后面使用子查询
  • 单行子查询—返回的数据只有一条，反之就是多条子查询
  • any语句 取其中的一个值进行比较 select * from a ehere money > any(selsect money from b where c= 30) 只要大于集合的全部返回。
  • all 取其中所有值进行比较，用法同上。
  • 相关子查询：将主查询的值传递给子查询。
