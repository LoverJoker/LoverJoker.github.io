创建Dao子模块

  1. 在ssh-parent项目上右击 ，创建时选择 Maven Module
  2. 填写子模块名称ssh-dao
  3. packing那里选择jar
  4. 把属于dao的代码拷贝到 该模块中
  5. 完成后发布到本地仓库中
